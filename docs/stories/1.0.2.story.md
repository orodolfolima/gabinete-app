# Story 1.0.2: CI/CD Pipeline

## 📋 Contexto

Configurar automação de testes, build e deploy usando GitHub Actions. Garante que código passeia por quality gates antes de ser mergeado.

## 🎯 Objetivo

Estabelecer pipeline de CI/CD que:
1. Valida código em PRs (lint, typecheck, testes)
2. Constrói Docker images em develop
3. Faz push para Docker Hub em main
4. Rastreia cobertura de testes com Codecov

## ✅ Acceptance Criteria (Detalhado)

### Workflow 1: PR Validation (.github/workflows/pr.yml)

**Trigger:** `on: [pull_request]`

```yaml
jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm install
      - run: npm run lint

  typecheck:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm install
      - run: npm run typecheck

  test:
    runs-on: ubuntu-latest
    services:
      postgres:
        image: postgres:15
        env:
          POSTGRES_PASSWORD: postgres
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
        ports:
          - 5432:5432
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm install
      - run: npm test -- --coverage
      - uses: codecov/codecov-action@v3
        with:
          files: ./coverage/lcov.info
          fail_ci_if_error: true
          flags: unittests
          name: codecov-umbrella
```

### Workflow 2: Build on Develop (.github/workflows/build.yml)

**Trigger:** `on: push branches: [develop]`

```yaml
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: docker/setup-buildx-action@v2
      - run: docker build -t app:${{ github.sha }} .
      - run: docker save app:${{ github.sha }} | gzip > image.tar.gz
      - uses: actions/upload-artifact@v3
        with:
          name: docker-image
          path: image.tar.gz
```

### Workflow 3: Push to DockerHub on Main (.github/workflows/release.yml)

**Trigger:** `on: push branches: [main]`

```yaml
jobs:
  push:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: docker/login-action@v2
        with:
          username: ${{ secrets.DOCKER_USERNAME }}
          password: ${{ secrets.DOCKER_PASSWORD }}
      - uses: docker/build-push-action@v4
        with:
          push: true
          tags: |
            ${{ secrets.DOCKER_USERNAME }}/siggap:latest
            ${{ secrets.DOCKER_USERNAME }}/siggap:${{ github.sha }}
```
